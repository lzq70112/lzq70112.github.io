判断 192.168.0.0/24 网络里，当前在线的 IP 有哪些，能 ping 通则认为在线
<!-- more -->
## 脚本如下
```shell
#!/bin/bash

for ip in `seq 1 255`;do
    ping -c 1 192.168.0.$ip > /dev/null 2>&1
    if [ $? -eq 0 ];then
        echo "192.168.0.$ip UP"
    else
        echo "192.168.0.$ip DOWN"
    fi
done
```
